# Reto Semana 02 - Conversión de Temperaturas

## Descripción

Este programa procesa un archivo CSV con temperaturas de distintas ciudades. Convierte todas las temperaturas a Celsius, clasifica el clima y genera un reporte estandarizado.

## Uso

El programa lee datos desde la entrada estándar (stdin) y escribe el resultado en la salida estándar (stdout).

Ejecución:

```bash
python main.py < archivo_entrada.txt
